联通网盘统一服务 - 部署清单
=====================================

📁 必需文件:
├── simple_unified_service.py    # 主服务文件
├── requirements.txt             # Python依赖
├── tokens.json                  # Token配置文件
└── 部署说明.md                  # 详细部署文档

🚀 启动脚本:
├── start.sh                     # Linux/Mac启动脚本
└── start.bat                    # Windows启动脚本

🐳 Docker部署 (可选):
├── Dockerfile                   # Docker镜像配置
└── docker-compose.yml           # Docker编排配置

📋 快速部署步骤:

1. 安装Python 3.7+
2. 安装依赖: pip install -r requirements.txt
3. 配置Token: 编辑 tokens.json
4. 启动服务: 
   - Windows: 双击 start.bat
   - Linux/Mac: bash start.sh
   - 手动: python simple_unified_service.py

🔐 默认登录信息:
- 用户名: admin
- 密码: 3150261994
- 地址: http://localhost:8000

📋 主要功能:
✅ Token负载均衡管理
✅ 联通网盘文件下载API
✅ 实时文件列表获取
✅ Web管理界面
✅ API接口服务

🔧 生产环境建议:
- 使用 gunicorn 启动: pip install gunicorn
- 配置 Nginx 反向代理
- 设置 systemd 服务
- 启用 HTTPS
- 修改默认密码

📞 技术支持:
- 查看日志排查问题
- 检查Token有效性
- 确认网络连接
- 验证依赖库版本
